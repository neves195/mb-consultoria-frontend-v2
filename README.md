# MB Consultoria — Sistema de Gestão de Treinamentos (v2)

Sistema web desenvolvido para automatizar a gestão de treinamentos profissionalizantes
e Normas Regulamentadoras (NRs) da MB Consultoria e Treinamento Neves.

Esta é a **versão 2** do front-end, reconstruída do zero com design system profissional,
componentes reutilizáveis e foco em experiência do usuário.

---

## Sobre o Projeto

A MB Consultoria é uma empresa especializada em treinamentos de segurança do trabalho
(NRs). Este sistema substitui processos manuais por uma plataforma digital centralizada,
desenvolvida por um estudante de 2° ano de Sistemas de Informação como projeto real.

---

## Funcionalidades

- 🔐 Autenticação segura com JWT
- 📊 Dashboard analítico com gráficos interativos e filtros de período
- 🏢 Gestão completa de empresas clientes
- 👤 Cadastro e gestão de alunos com histórico de NRs
- 📚 Catálogo de cursos e Normas Regulamentadoras
- 📋 Gestão de treinamentos com controle de vagas
- 📅 Agenda visual com calendário mensal interativo
- ✅ Matrículas com controle de progresso
- 📄 Geração automática de certificados em PDF
- 💰 Módulo financeiro com controle de cobranças
- ⚙️ Configurações do sistema e perfis de acesso
- 🌙 Tema escuro/claro
